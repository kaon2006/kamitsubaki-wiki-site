# HiFi-P

HiFi-P は、YouTube や TikTok で活動するサウンドクリエイター。Hi-Fi トラックと印象的なメロディーで知られ、オンライン音楽コミュニティで広い支持を集めている。

「Musical Isotope」ブランドのもとで楽曲制作・リリースを行い、多数のアーティストへ楽曲提供を行っている。代表作には、KAF x KAFU への提供曲「流水線メーデー」（Ryuusenkei Mayday）や、ASU への提供曲「Imperfect」が挙げられる。これらの作品は、幅広いスタイルを自在に操る彼の創作力を示している。

HiFi-P は KAMITSUBAKI STUDIO CREATOR FARM に所属し、同レーベルの音楽クリエイターチームの重要な一員として活動している。

イラストは Ritsao（りたお）が担当。
