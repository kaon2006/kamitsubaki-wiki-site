# 2024 Logs

Use this folder for 2024 news and observation records.

Recommended frontmatter fields:

```yaml
date: "2024.06.01"
type: "UPDATE"
title: "Log title"
summary: "Short card summary"
order: 1
```

The `order` field controls homepage display order. It does not have to match strict chronology if the wiki needs a curated narrative flow.
