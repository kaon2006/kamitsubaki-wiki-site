# Log Content

Logs are grouped by year and each record is stored as Markdown.

Example:

```text
2024/2024-06-01-vwp-live/zh.md
2024/2024-06-01-vwp-live/ja.md
2024/2024-06-01-vwp-live/en.md
```

Each file can power both the `03. LOG` news card and the corresponding detail page:

```text
/{locale}/logs/2024/2024-06-01-vwp-live
```

Use date-prefixed `translationKey` values so files stay readable and sortable.
